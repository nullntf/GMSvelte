# GMSvelte

Gestor Multinegocio con Svelte para inventario, ventas y más.

## Tecnologías

- **Frontend:** Svelte 5, SvelteKit, TypeScript
- **Backend:** Node.js con adapter-node
- **Base de datos:** MySQL
- **Deploy:** Railway
- **Herramientas:** ESLint, Prettier, MCP Servers

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run start
```

## MCP Servers Configurados

- Svelte (documentación y análisis)
- Filesystem (acceso seguro al proyecto)
- Git (operaciones de control de versiones)
- MySQL (consultas a base de datos)
- Railway (gestión de deploy)
